## 2. 개요

- **JAVA**로 구현한 **도서 추천 사이트**입니다.
- **Oracle DB**에서 미리 저장된 책정보를 불러오고 사용자들은 **댓글**로 평가를 할 수 있습니다.

## 3. 특징

- **BootStrap5**를 최대한 활용하여 css, js 코드를 최소화 하였습니다.
- **로그인, 회원가입, 검색, 리스트, 상세페이지, 댓글** 기능을 지원합니다.
- **댓글 입력**은 **회원전용** 기능이며 본인 댓글만 **수정/삭제**가 가능합니다.
- **댓글 별점**이 실시간으로 반영됩니다.
